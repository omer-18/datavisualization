# DataViz Studio

A drag-and-drop data visualization tool built with React, TypeScript, and Vega-Lite.



## Live Demo

Try the application yourself: **https://zero2sudo.github.io/datavisualization/**

## Features

- Drag-and-drop interface for building charts
- Automatic field type detection (quantitative, nominal, temporal)
- Real-time Vega-Lite chart rendering
- Support for X, Y, Color, and Size encodings

This app is like a visual chart builder. You have a dataset (like car information), and the app shows you all the columns (miles per gallon, car brand, etc.). You drag those columns into slots like "X-axis" or "Y-axis" or "Color," and it automatically builds a chart. It figures out whether your data is numbers, categories, or dates, and picks the right chart type. No codingâ€”just drag, drop, and see your chart appear. It's like a smart version of Excel's chart wizard, but more interactive and visual.
